## 🛠 Install

### npm or yarn

Install the `@zk-kit/eddsa-poseidon` package and its peer dependencies with npm:

```bash
npm i @zk-kit/eddsa-poseidon
```

or yarn:

```bash
yarn add @zk-kit/eddsa-poseidon
```
